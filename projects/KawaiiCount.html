What is Kawaii Count?

Kawaii Count is a Restaurant Inventory System application that is designed for cafes or small coffee shops. The app combines functionality such as an analytics page, an inventory page, a Menu system, and a login and sign-up page for employees and administrators to access the application.

This application is a joint collaboration project of two major subjects, where our group is tasked to create a restaurant Inventory System using Java and its tools, such as Maven and JavaFX, to create the application, and PostgreSQL to create the database.
