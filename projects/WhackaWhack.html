WhackAWhack

A meme whack-a-mole style arcade game!

What is WhackAWhack?

WhackAWhack is java game application that takes inspiration from Uncanny cat golf game and whack-a-mole. The game is created using maven and JavaFX with scenebuilder

This is a Java finals project where we are tasked to create a java application using multithreading and Java tools. I chose a game application because I love playing video games and I want to improve my Java programming skills.

What I would improve and add

Im planning to add a options feature and game menu. Im also planning to add more types of moles to make the gameplay loop more fun than ever. I would also improve the level design of my game where there is currently one level and one difficulty.
